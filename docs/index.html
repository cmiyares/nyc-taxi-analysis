NYC Taxi Analysis

A data analysis project exploring NYC taxi trip patterns, fares, and demand trends using Python, pandas, and data visualization.

View GitHub Repo Refresh Site

About the Project

This project analyzes NYC taxi trip data to better understand ride patterns, trip volume, fare behavior, and overall transportation trends across the city.

Tools Used

  • Python
  • pandas
  • Matplotlib
  • Jupyter Notebook

What I Explored

  • Trip distribution and demand patterns
  • Relationships between fare amount and trip distance
  • Trends that help explain taxi usage across NYC
